ส่งข้อมูลมาทดสอบได้ที่: (รับทุก method: GET, POST, PUT, DELETE ฯลฯ และทุก path ย่อย เช่น /hook/order/create)

ยังไม่มี request เข้ามา
ลองยิงข้อมูลมาที่ hook URL ด้านบน

เลือก request จากรายการด้านซ้ายเพื่อดูรายละเอียด

ผลลัพธ์ response จะแสดงที่นี่

เลือกรูปภาพเพื่อดู metadata (วันที่ถ่าย, พิกัด GPS, ที่อยู่, ข้อมูลกล้อง)

📤 แนบรูปจากเครื่องนี้: เปิดหน้านี้จากมือถือในวง LAN เดียวกัน (URL เดียวกัน) แล้วแนบรูป — รูปจะเด้งขึ้นทุกจอที่เปิดแท็บนี้ทันที หรือให้ mobile app ยิง multipart มาที่

ยังไม่มีไฟล์ส่งเข้ามา
แนบรูปจากมือถือแล้วจะแสดงที่นี่ทันที

เลือกไฟล์จากรายการด้านซ้ายเพื่อดูรายละเอียดและ metadata
