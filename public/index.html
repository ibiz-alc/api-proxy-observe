🔌 เชื่อม Proxy Postern มาที่ — Host … Port 8888
🔌 USB
เสถียร/เร็วสุด — Host 127.0.0.1 · Port 8888
1. เสียบสาย + เปิด USB debugging   2. บน Mac รัน ./start.sh (ตั้ง adb reverse tcp:8888 tcp:8888 ให้เอง)   3. ในแอปกรอก host/port ข้างบน → เชื่อมต่อ
📶 Wi-Fi
มือถืออยู่วง Wi-Fi เดียวกับ Mac — Host … · Port 8888
1. มือถือต่อ Wi-Fi วงเดียวกับ Mac   2. ในแอปกรอก Host = IP ข้างต้น, Port 8888 → เชื่อมต่อ   (Wi-Fi บางที่ (office/สาธารณะ) บล็อกพอร์ต/กันเครื่องคุยกัน — ถ้าต่อไม่ได้ใช้ USB)
ทั้ง 2 วิธี: ติดตั้ง CA ของ mitmproxy บนมือถือครั้งเดียว (ปุ่ม "ติดตั้ง CA" ในแอป) — ต้องต่อ Mac ให้ mitmproxy รันอยู่ (./start.sh)
📲 เชื่อม Proxy Postern จากเว็บ (USB)
กำลังโหลด device…
วิธีทำให้เห็นเนื้อใน HTTPS:
  1. บนมือถือ ตั้งค่า Wi-Fi → HTTP Proxy → Manual → Server = , Port =
  2. เปิดเบราว์เซอร์บนมือถือ ดาวน์โหลด CA cert จาก
  3. iOS: Settings → General → VPN & Device Management → ติดตั้ง profile → จากนั้น Settings → General → About → Certificate Trust Settings → เปิด "Enable Full Trust" ให้ cert นี้
  4. Android: Settings → Security → Install a certificate → CA certificate (แอปบางตัวต้องตั้ง network_security_config เพิ่มถึงจะเชื่อ user CA)
  5. เสร็จแล้วเปิดแอป/เว็บบนมือถือ ทราฟฟิกจะโผล่ด้านล่างแบบ real-time

⚠️ CA cert นี้ให้เครื่องเชื่อ proxy ของเราถอดรหัส HTTPS ได้ ใช้เฉพาะบนอุปกรณ์ทดสอบของคุณเอง และถอนการติดตั้งเมื่อเลิกใช้
